# YouTube Clone

A YouTube clone built with React, designed to simulate the core features of the original platform. This project serves as both a learning exercise and a functional web application.

## Live Demo

Check out the live demo [here](https://velvety-paletas-da07fa.netlify.app/).

## Features

- Video streaming and playback
- Responsive design
